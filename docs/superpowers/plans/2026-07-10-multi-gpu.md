# Multi-GPU Scheduling Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rewrite `gq` for multi-GPU scheduling: `gq add --gpus N 'cmd'` declares card count, the daemon runs multiple jobs concurrently on different cards, picks N idle cards per job and injects `CUDA_VISIBLE_DEVICES`. Single-card jobs (no `--gpus`) work as before with a notice.

**Architecture:** `gpu_is_idle()->bool` becomes `busy_cards()->set[int]`. The daemon loop changes from blocking `_run_job` to concurrent Popen tracking: launch the queue head when >=N idle cards exist, reap finished jobs each poll cycle. `state.json`'s `running` becomes a dict keyed by job id (each with `cards`, `n`, `pid`). `gq stop <id>` stops a specific job. Crash recovery loops over all running entries. Second Ctrl-C kills all running jobs.

**Tech Stack:** Python 3 stdlib (`os`, `signal`, `subprocess`, `json`); pytest (with `PYTHONPATH=` prefix on this machine); bash completion tests.

## Global Constraints

- **Zero external dependencies** — pure Python stdlib.
- **Single-file script** — production changes in `gq` at repo root; tests in `tests/test_gq.py`.
- **Full replacement** — no `--multi` flag, no two code paths. Single-card degrades naturally (N=1).
- **`--gpus N` is the card-count source** — do NOT parse the command string. N is stored on the job at add time. No `--gpus` -> n=1 + notice. No escape hatch (all jobs get `CUDA_VISIBLE_DEVICES` injected).
- **Strict FIFO** — no head-of-line skipping. If head needs N cards and fewer than N idle, wait.
- **Idle = no compute (type=C) process of the current user on that card** (unchanged semantics; other users don't block, G-type ignored, nvidia-smi failure -> all busy).
- **`gq stop <id>` requires an argument** — no-arg is an error.
- **Tests run with `PYTHONPATH=` prefix** on this machine (ROS pytest plugin conflict — see memory `ros-pytest-workaround`). Completion tests run with plain `bash tests/test_completion.sh`.
- **Current branch:** `main` — branch to `feat/multi-gpu` at execution time.
- **env-capture feature is preserved** — `gq add` still snapshots `os.environ` into `job["env"]`. `CUDA_VISIBLE_DEVICES` is added to that snapshot at launch time.

```python
_TOTAL_CARDS = None  # type: Optional[int]


def _total_cards() -> int:
    """Count GPUs via nvidia-smi -L. Cached after first call."""
    global _TOTAL_CARDS
    if _TOTAL_CARDS is not None:
        return _TOTAL_CARDS
    try:
        result = subprocess.run(
            ["nvidia-smi", "-L"],
            capture_output=True, text=True, timeout=10
        )
        # Each GPU is one line: "GPU 0: NVIDIA GeForce ..."
        _TOTAL_CARDS = sum(1 for line in result.stdout.splitlines()
                           if line.startswith("GPU "))
        if _TOTAL_CARDS == 0:
            _TOTAL_CARDS = 1  # fallback so idle math doesn't divide weirdly
    except (FileNotFoundError, subprocess.TimeoutExpired):
        _TOTAL_CARDS = 1
    return _TOTAL_CARDS


def busy_cards() -> set:
    """Return the set of physical GPU indices with a compute (type=C) process
    owned by the current user. Graphics (type=G) and other users' processes
    do not count. On nvidia-smi failure/timeout, returns ALL card indices
    (safe default: daemon treats all as busy -> no launch).
    """
    uid = os.getuid()
    total = _total_cards()
    try:
        result = subprocess.run(
            ["nvidia-smi", "pmon", "-s", "um", "-c", "1"],
            capture_output=True, text=True, timeout=10
        )
    except FileNotFoundError:
        print("[gq] ERROR: nvidia-smi not found. Is the NVIDIA driver installed?",
              file=sys.stderr)
        sys.exit(1)
    except subprocess.TimeoutExpired:
        print("[gq] WARNING: nvidia-smi timed out, treating all GPUs as busy.",
              file=sys.stderr)
        return set(range(total))

    if result.returncode != 0:
        print(f"[gq] WARNING: nvidia-smi failed (exit {result.returncode}), "
              "treating all GPUs as busy.", file=sys.stderr)
        return set(range(total))

    busy = set()
    for line in result.stdout.splitlines():
        if line.startswith("#") or not line.strip():
            continue
        parts = line.split()
        if len(parts) < 3:
            continue
        if parts[2] != "C":          # skip Graphics (G)
            continue
        try:
            gpu = int(parts[0])
            pid = int(parts[1])
            if os.stat(f"/proc/{pid}").st_uid == uid:
                busy.add(gpu)
        except (ValueError, FileNotFoundError, PermissionError):
            continue
    return busy
```

```python
def read_state() -> dict:
    default = {"daemon_pid": None, "running": {}}
    state = _lock_and_read(STATE_FILE, default)
    if not isinstance(state, dict):
        print("[gq] WARNING: state.json corrupted (non-object), resetting.", file=sys.stderr)
        return dict(default)
    running = state.get("running")
    # New shape: running is a dict keyed by job id (possibly empty).
    # Old shape: running was None or a single job dict with "id". Migrate.
    if running is None or (isinstance(running, dict) and "id" in running and "pid" in running
                           and not any(isinstance(v, dict) for v in running.values())):
        state["running"] = {}
    elif not isinstance(running, dict):
        state["running"] = {}
    return {**default, **state}
```

```python
def _launch_job(job: dict, cards: list) -> Optional[subprocess.Popen]:
    """Launch a job on the given cards (non-blocking). Returns the Popen, or
    None on launch failure. Injects CUDA_VISIBLE_DEVICES into the job's env.
    """
    cwd = job["cwd"]
    if not os.path.isdir(cwd):
        print(f"[gq] WARNING: cwd '{cwd}' does not exist, skipping job {job['id']}.")
        return None
    env = dict(job.get("env") or {})
    env["CUDA_VISIBLE_DEVICES"] = ",".join(str(c) for c in cards)
    print(f"[gq] >>> {job['cmd']}   (GPU {','.join(str(c) for c in cards)})")
    try:
        proc = subprocess.Popen(job["cmd"], shell=True, cwd=cwd,
                                start_new_session=True, env=env)
    except Exception as exc:
        print(f"[gq] ERROR running job {job['id']}: {exc}")
        return None
    return proc
```

```python
_shutdown_requested = False
_running = {}  # type: dict  # {id: {"proc": Popen, "job": dict, "cards": [int], "start": float}}


def _daemon_loop(poll_interval: int) -> None:
    """Concurrent multi-GPU polling loop. Launches the queue head when enough
    idle cards exist; reaps finished jobs each cycle. Exits on shutdown."""
    global _shutdown_requested
    _shutdown_requested = False

    ctrl_c_count = [0]

    def _handle_sigint(sig, frame):
        ctrl_c_count[0] += 1
        if ctrl_c_count[0] == 1:
            print("\n[gq] Ctrl-C received. Waiting for running jobs to finish. "
                  "Press Ctrl-C again to force-kill all.", file=sys.stderr)
            _shutdown_requested = True
        else:
            # Second Ctrl-C: SIGKILL ALL running jobs' process groups.
            for rid, info in list(_running.items()):
                pid = info["proc"].pid
                try:
                    os.killpg(os.getpgid(pid), signal.SIGKILL)
                except ProcessLookupError:
                    pass
            sys.exit(0)

    old_handler = signal.signal(signal.SIGINT, _handle_sigint)
    try:
        while not _shutdown_requested:
            # 1. Reap finished jobs.
            for rid in list(_running.keys()):
                info = _running[rid]
                code = info["proc"].poll()
                if code is not None:
                    elapsed = _format_elapsed(time.monotonic() - info["start"])
                    if code == 0:
                        print(f"[gq] <<< job {rid} DONE (exit=0) in {elapsed}")
                    else:
                        print(f"[gq] <<< job {rid} FAILED (exit={code}) in {elapsed}")
                    del _running[rid]
                    _state = read_state()
                    _state.setdefault("running", {}).pop(rid, None)
                    write_state(_state)

            # 2. Try to launch the queue head if enough idle cards.
            total = _total_cards()
            busy = busy_cards()
            idle = set(range(total)) - busy
            # Cards already assigned to running jobs are not idle even if
            # nvidia-smi hasn't caught up.
            assigned = set()
            for info in _running.values():
                assigned.update(info["cards"])
            idle = idle - assigned

            with _locked_queue() as q:
                if q:
                    head = q[0]
                    n = head.get("n", 1)
                    if len(idle) >= n:
                        q.pop(0)
                        cards = _pick_idle(idle, n)
                        proc = _launch_job(head, cards)
                        if proc is not None:
                            head["started_at"] = datetime.datetime.now().isoformat(timespec="seconds")
                            _running[head["id"]] = {"proc": proc, "job": head,
                                                    "cards": cards, "start": time.monotonic()}
                            _state = read_state()
                            _state.setdefault("running", {})[head["id"]] = {
                                **head, "cards": cards, "pid": proc.pid
                            }
                            write_state(_state)
                            print(f"\n[gq] GPU idle — starting job {head['id']} on "
                                  f"GPU {','.join(str(c) for c in cards)}")

            time.sleep(poll_interval)
    finally:
        signal.signal(signal.SIGINT, old_handler)

    print("[gq] daemon exiting.")
```

```python
def cmd_stop(args) -> None:
    """Stop a specific running job by ID (SIGKILL its process group).

    With multiple jobs running concurrently, an ID is required. The daemon
    stays alive and reaps the killed job, releasing its cards.
    """
    state = read_state()
    running = state.get("running") or {}
    job_id = args.job_id

    matches = [jid for jid in running if jid.startswith(job_id)]
    if not matches:
        print(f"[gq] no running job matches '{job_id}'.")
        return
    if len(matches) > 1:
        ids = ", ".join(matches)
        print(f"[gq] ambiguous prefix '{job_id}' matches: {ids}. Be more specific.")
        return

    target_id = matches[0]
    rpid = running[target_id].get("pid")
    if not rpid:
        print(f"[gq] job {target_id} has no pid yet (not fully started).")
        return
    try:
        pgid = os.getpgid(rpid)
        os.killpg(pgid, signal.SIGKILL)
    except ProcessLookupError:
        print(f"[gq] job {target_id} already finished (pid {rpid} not found).")
        return
    except PermissionError as exc:
        print(f"[gq] ERROR: cannot stop job {target_id} (pid {rpid}): {exc}")
        return
    print(f"[gq] stopped job {target_id} (pid {rpid}).")
```
